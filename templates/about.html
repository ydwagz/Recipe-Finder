About

Recipe Finder

Recipe Finder is a simple place to save recipes, search them by ingredients, and keep cooking ideas easy to find later.

Users can create an account, upload recipes with photos, list ingredients, and write the full cooking details. Each recipe stays connected to the person who uploaded it, so only that user or an admin can edit or remove it.

The search bar is built for real kitchen moments: type what you already have, like eggs, bread, cheese, rice, or tomatoes, and the site will show matching saved recipes from the collection.
